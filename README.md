恋みくじアプリ (最強版)

ボタンをクリックするだけで、今日の恋愛運を占うことができるシンプルなウェブアプリケーションです。

〇概要
このプロジェクトは、HTML、CSS、JavaScriptを使用して構築されたおみくじアプリです。ランダムに選ばれた運勢（大吉、中吉、小吉、凶）と、それに応じたアドバイスや絵文字が表示されます。

〇特徴
シンプル操作: 「おみくじを引く」ボタンを押すだけの簡単なUI。

ランダム機能: JavaScriptの Math.random() を利用したランダムな結果表示。

可愛いデザイン: 恋愛をイメージしたピンク基調のスタイル。

〇ファイル構成
プロジェクトは以下の3つのファイルで構成されています。

〇Plaintext
.
├── index.html   # アプリの構造（HTML）
├── style.css    # アプリの見た目（CSS）
└── script.js    # 占いロジック（JavaScript）
〇使い方
すべてのファイルを同じフォルダ（ディレクトリ）に保存します。

index.html をブラウザ（Google Chrome, Edgeなど）で開きます。

「おみくじを引く」ボタンをクリックすると、結果が表示されます。

〇技術詳細
HTML5: ページの骨組みを作成。

CSS3: 背景色（ピンク）やボタンのスタイリング、中央揃えなどのレイアウト。

JavaScript (ES6):

配列による結果リストの管理。

イベントリスナーによるクリックイベントの検知。

Math.floor と Math.random を組み合わせたランダムインデックスの生成。

〇カスタマイズ方法
運勢の種類を増やしたい場合は、script.js 内の fortunes 配列に新しい文字列を追加するだけで、自動的に反映されます。

JavaScript
const fortunes = [
    "大吉　💕　素敵な出会いがあるかも！",
    // ここに新しい運勢を追加
];
